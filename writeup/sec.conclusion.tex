Kritische Würdigung
\begin{itemize}
\item Threadproblematik (und damit verbundene Limitation bei den Experimenten -> Zeit und Entitätszahl))
\item Andere Einflüsse wie Medikamentation oder genetische Diversität / Immunisierung unberücksichtigt
\item Konfigurationen mit Strategie nach fest vorgegebenem Schema
\end{itemize}

Das genutzte Framework erzeugt für jede Entität einen neuen Thread, welchem die Java Virtual Machine einen bestimmten Teil des Arbeitsspeichers reserviert. So kommt es bei ausreichend großer Anzahl von Entitäten dazu, dass zu wenig Arbeitsspeicher vorhanden ist. Daher werden Teile auf die Festplatte ausgelagert, wodurch sich die Dauer einer Simulation stark vergrößert. 

Ein Imker könnte nach dem Bemerken einer Erkrankung seiner Bienen ein Medikament verabreichen, um einer weiteren Ausbreitung vorzubeugen. In der Natur besitzen einige Bienenstämme eine natürliche Immunität gegen das Virus oder können diese nach einer Infektion entwickeln. Sowohl die Verabreichung von Medikamenten, sowie eine eventuell vorhandene Immunität wurden in diesem Projekt nicht betrachtet. So könnte nach einer Ansteckung noch entschieden werden, ob eine Biene auch erkrankt. Desweiteren wäre es möglich die Bienen einzelner Bienenstöcke gegen die Krankheit zu immunisieren. Bisher werden die Bienenstöcke nach einem festen Schema platziert: Es gibt Gruppen von Stöcken, die mit größtmöglicher Entfernung untereinander platziert werden. Innerhalb einer Gruppe werden die Stöcke nah nebeneinander gesetzt. Dabei werden andere Anordnungen nicht simuliert. So sollten auch weitere geometrische oder zufällig gewählte Platzierungen überprüft werden. Als weiterer Indikator könnte die Ausbreitungsgeschwindigkeit der Krankheit innerhalb eines Stocks beobachtet werden. So könnte ein Zeitpunkt ermittelt werden, der zwangsläufig zum Absterben des Stocks führt. Möglicherweise wäre das der späteste Zeitpunkt zur effektiven Medikamentenverabreichung durch den Imker.

Ausblick / Erweiterungsmöglichkeiten / \textbf{oben mit abhandeln?}
\begin{itemize}
\item Einbezug von Immunisierung oder genetischer Diversität
\item genauere Untersuchung des Infektionsverlaufs (nicht nur Zeit bis alle tot sind sondern auch lokale Ausbreitungsgeschwindigkeit z.B.)
\end{itemize}

\section{Schlussbetrachtung}
Aus Ergebnissen des Experiments lässt sich schlussfolgern, dass tendenziell eine Platzierungsstrategie ausgewählt werden sollte, bei der sich die einzelnen Bienenstöcke möglichst weiter auseinander befinden. Ob sich dieses Ergebnis der Simulation auf die Realität übertragen lässt, ist schwer zu sagen. Für das Simulationsmodell mussten verschiedene Annahmen getroffen werden bzw. war eine naturgetreue Abbildung aus technischen Gründen nicht möglich.

Das genutzte Framework erzeugt für jede Entität einen neuen Thread, für welchen die Java Virtual Machine einen bestimmten Teil der Systemressourcen reserviert. So kommt es bei ausreichend großer Anzahl von Entitäten dazu, dass die Ressourcen überlastet werden. Daher kann sich die Dauer eines Simulationslaufes bei entsprechend großer Anzahl von Entitäten stark vergrößern.

Ein Imker könnte nach dem Bemerken einer Erkrankung seiner Bienen ein Medikament verabreichen, um einer weiteren Ausbreitung vorzubeugen. In der Natur besitzen einige Bienenstämme eine natürliche Immunität gegen das Virus oder können diese nach einer Infektion entwickeln. Sowohl die Verabreichung von Medikamenten, als auch eine eventuell vorhandene Immunität wurden in diesem Projekt nicht betrachtet. So könnte nach einer Ansteckung noch entschieden werden, ob eine Biene auch tatsächlich erkrankt. Desweiteren wäre es möglich die Bienen einzelner Bienenstöcke gegen die Krankheit zu immunisieren.

Bisher werden die Bienenstöcke nach einem festen Schema platziert: Es gibt Gruppen von Stöcken, die mit größtmöglicher Entfernung untereinander platziert werden. Innerhalb einer Gruppe werden die Stöcke nah nebeneinander gesetzt. Dabei werden andere Anordnungen nicht simuliert. So sollten auch weitere geometrische Formen oder zufällig gewählte Platzierungen überprüft werden.

Als weiterer Indikator könnte die Ausbreitungsgeschwindigkeit der Krankheit innerhalb eines Stocks dienen. So könnte ein Zeitpunkt ermittelt werden, der zwangsläufig zum Absterben des Stocks führt. Möglicherweise wäre das der späteste Zeitpunkt zur effektiven Medikamentenverabreichung durch den Imker.
